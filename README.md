# Geesala Sandhyarani — Professional Portfolio Website

Welcome to my personal portfolio repository! This is a premium, high-fidelity single-page website showcasing my academic timeline, internships, independent software projects, professional certifications, and active coding profiles.

The visual layout features a deep obsidian dark mode theme inspired by modern web agency aesthetics, highlighted by glassmorphism card UI structures, floating interactive navigations, and customized hover elements.

Live Link: [https://github.com/GeesalaSandhyarani/portfolio](https://portfolio-beryl-seven-95.vercel.app/)

---

## 🚀 Key Features

*   **Interactive Glassmorphic Design**: Built using custom Vanilla CSS layout structures (`backdrop-filter: blur(16px)`) featuring thin borders and soft inset dropshadows.
*   **Dynamic Custom Cursor**: A custom dual-element dot-and-ring pointer tracking mouse coordinates with lag dampening, which expands and glows over clickable buttons.
*   **Dual Timelines Grid**: Aligns **Internship Work** (Web Development at Oasis Infobyte and Data Analysis at ApexPlanet Software) and **Education History** side-by-side on desktop viewports.
*   **Coding Profiles Section**: Seamlessly routes visitors to official developer dashboards including **LeetCode**, **CodeChef**, **GeeksforGeeks**, **Coddy**, **Kaggle**, **HackerRank**, and **Naukri Code360**.
*   **Case Study Modals**: Open detailed write-ups (Problem, Solution, UX Flow, Impact) and repository source code links dynamically without pushing users off the page.
*   **Responsive Layout**: Tailored using viewport breakpoints (grid and flex wrap configurations) to display beautifully on Desktop, Tablet, and Mobile screens.

---

## 🛠️ Technology Stack

*   **Markup**: Semantic HTML5 structures.
*   **Styling**: Vanilla CSS3 custom variables, radial glow animations, scroll observers, and grid systems.
*   **Interactions**: Vanilla ES6 JavaScript (Intersection Observer API for scroll highlights, custom cursor tracking, client-side validation logic).
*   **Assets**: FontAwesome icon sets, Google Fonts integration (Outfit & Inter), and high-resolution layout assets.

## 💻 Local Setup & Run

To open and run the portfolio locally, you do not need complex installation scripts. 

### Option A: Open directly
Simply clone the repository and double-click `index.html` to run in any browser.

### Option B: Local Web Server (Python)
To test observers, smooth navbar capsule scrollspy transitions, and modal load features on a local host:
1. Clone the project:
   ```bash
   git clone https://github.com/GeesalaSandhyarani/portfolio.git
   cd portfolio
   ```
2. Start a simple server:
   ```bash
   python -m http.server 8080
   ```
3. Open `http://localhost:8080` in your web browser.
